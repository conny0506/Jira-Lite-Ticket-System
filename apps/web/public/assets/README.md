Asset library for web app visuals.

Recommended structure:
- `assets/icons/site-icon.jpg` -> browser tab icon (favicon source)
- `assets/icons/apple-touch-icon.png` -> iOS home icon
- `assets/brand/logo-primary.png` -> main logo
- `assets/brand/logo-mark.png` -> compact logo mark
- `assets/brand/og-image.png` -> social preview image
- `assets/illustrations/*` -> UI illustrations

Usage examples:
- `<img src="/assets/brand/logo-primary.png" alt="Logo" />`
- CSS: `background-image: url('/assets/illustrations/hero-bg.png');`
